###### Wireless security tool 

## About
**Wireless security tool** is a Linux program that automates wireless auditing by using popular tools like **Wireshark (tshark)** and the **Aircrack-ng Suite**.

2. Wireshark (tshark)
3. Aircrack-ng Suite
